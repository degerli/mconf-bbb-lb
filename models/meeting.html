Model Meeting

Has a class Meeting to represent meetings and a database (a simple in-memory object) to store the meetings saved.

fs = require("fs") Server = require("./server")

The database of meetings.

We only need this password for the mobile client. See routes/mobile.sendGetMeetingInfoToAll().
